## 9. Data Security

We implement the following security measures:

- All data in transit is encrypted via **HTTPS / WSS (TLS 1.2+)**
- Passwords are stored using **bcrypt hashing with salt**
- Database access is restricted to the application server only (no public exposure)
- Sessions use signed, short-lived JWT tokens
- All game actions are logged immutably for dispute resolution
- Security incidents are investigated and resolved promptly

**Data breach notification:** In the event of a breach affecting your personal data, we will notify affected users and relevant authorities within **72 hours** of becoming aware of the breach, in accordance with applicable regulations.
