## Quick Summary

The TzDraft tournament system supports:
- **Scopes:** Global, Country-wide, Regional
- **Eligibility filters:** ELO range, career wins, AI level beaten, AI level played
- **Formats:** Single Elimination, Round Robin (Phase 1) → Swiss, Double Elimination (later)
- **Styles:** Blitz, Rapid, Classical, Unlimited
- **37 new files, 5 modified** across backend + frontend
